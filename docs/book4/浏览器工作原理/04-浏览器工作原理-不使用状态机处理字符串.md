# 浏览器工作原理-不使用状态机处理字符串

## 使用有限状态机处理字符串

- 在一个字符串中，找到字符'ab'

```
  function match(string) {
    let flag = false
    for (let key of string) {
      if (flag && key === 'b') {
        return true
      } else {
        flag = false
      }
      if (key === 'a') {
        flag = true
      }
    }
    return false
  }
  match('acv')
  match('abcd')
```
